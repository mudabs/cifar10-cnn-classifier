# CIFAR-10 CNN - Computer Vision Assignment

## 1. What Runs First

1. Run data preparation:

```bash
python prepare_data.py
```

This downloads CIFAR-10 and organizes it into `data/train/` and `data/test/`.

2. Run the main entry point:

```bash
python main.py
```

Then choose from the menu:
- `1` Train
- `2` Evaluate
- `3` Visualize

You can also run `train.py`, `evaluate.py`, or `visualize.py` directly.

## 2. Where the Dataset Is Stored

```text
data/
|- train/   (50,000 PNG images across 10 class folders)
`- test/    (10,000 PNG images across 10 class folders)
```

Class folders:
`airplane`, `automobile`, `bird`, `cat`, `deer`, `dog`, `frog`, `horse`, `ship`, `truck`

## 3. How the Model Was Trained

- Architecture: Custom CNN with 3 ConvBlocks (`32 -> 64 -> 128` channels), depthwise-separable + standard convolutions, BatchNorm, ReLU
- Optimizer: Adam (`lr = 0.001`)
- Loss: CrossEntropyLoss
- Batch size: `64`
- Epochs: `10`
- Saved weights: `model.pth`

## 4. Output Files That Matter Most

- `model.pth`: Trained model weights (required for evaluation and visualization)
- `confusion_matrix.png`: Test-set confusion matrix generated by `evaluate.py`
- `visualizations/`: Feature-map PNGs generated by `visualize.py`
